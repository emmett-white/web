## Дескрипција

Овај колбек се позива када се филтерскрипта покренула

## Примери

```c
public OnFilterScriptInit()
{
    print("\n--------------------------------------");
    print("Филтерскрипта покренута.");
    print("--------------------------------------\n");
    return 1;
}
```
