## Дескрипција

Овај колбек се позива када се филтерскрипта стопирала/зауставила.

## Примери

```c
public OnFilterScriptExit()
{
    print("\n--------------------------------------");
    print(" Филтерскрипта заустављена");
    print("--------------------------------------\n");
    
    return 1;
}
```
